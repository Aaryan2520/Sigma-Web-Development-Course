## Conditional Statement

```bash

    console.log("You can drive")
}

    console.log("Are you kidding ?")
}

    console.log("You cannot drive")
}

```

## Assignment Operator

`=` x = y

`+=` x = x + y

`-=` x = x - y

`*=` x = x \* y

`/=` x = x / y

`%` x = x % y

`**` x = x \*\* y

## Comparison Operator

`==` Equal to

`!=` Not Equal to

`===` Equal Value And Type

`!==` Not Equal Value Or Not Equal Type

`>` Greater Than

`<` Less Than

`>=` Greater Than Or Equal To

`<=` Less Than Or Equal To

`?` Ternary Operator

## Ternary Operator

```bash
let a = 6
let b = 8
let c = a > b ? (a - b) : (b - a)
```

`let c = a > b ? (a - b) : (b - a)`
This Means

```bash
if (a > b) {
    let c = a - b
}
else {
    let c = b - a
}
```
